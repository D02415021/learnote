# 演示

> 查询最近暴力破解服务器密码的IP归属地

```shell
cat /var/log/secure | awk '/Failed/ {print $(NF-3)}' > ip_list.txt
```

然后查询

```shell
python ip_query.py ip_list.txt out.csv
```

就是利用淘宝IP地址查询API挨个把IP地址归属地查询一遍，结果放在`csv`文件中，`Excel`打开长这样
